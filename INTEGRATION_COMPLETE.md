## 💡 Customization Tips

### **Change Brand Colors**

Edit `style-premium.css` (lines 5-30):
```css
:root {
    --primary-dark: #0F1419;
    --neon-cyan: #00F0FF;
    --neon-blue: #0EA5E9;
    --accent-purple: #8B5CF6;

```

Edit `style-premium.css`:
```css
--transition: all 0.3s cubic-bezier(...);  /* Change 0.3 to desired speed */
```

## 🐛 Troubleshooting

### **3D Hero Not Showing**
- Check DevTools Console for errors
- Verify Three.js CDN loaded in Network tab
- Try different browser (Chrome recommended)
- Check GPU drivers are updated

### **Animations Laggy**
- Open DevTools → Performance tab
- Record while scrolling
- Check FPS - should be 60+
- Reduce animation duration if needed

### **CSS Not Applying**
- Hard refresh: Ctrl+Shift+R
- Clear browser cache
- Check file paths in HTML
- Verify file names match exactly

### **Scripts Not Loading**
- Open DevTools → Console tab
- Look for 404 errors in Network tab
- Verify script src attributes match filenames
- Check file paths are correct
